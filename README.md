# Crypto Porfolio project (Binance)  
  
## Стек проекта  
  
1. Vite + React + TypeScript  
2. reduxjs/toolkit  
3. Axios  
4. Framer-motion  
5. GravityUI в кач-ве UI компонентов  
  
## Для запуска проекта  
```
npm install  
  
npm run start  
```
